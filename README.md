# Where Did My Money Go?

> Developed by **@jdiassdev**

---

API backend para **controle financeiro pessoal**, focada em registrar entradas, saídas e entender **para onde o dinheiro está indo** ao longo do tempo.

Projeto pensado para praticar **Spring Boot na vida real**, com arquitetura limpa, CRUD bem feito e integração com banco relacional.

---

## Objetivo

- Registrar movimentações financeiras (entradas e saídas)
- Categorizar gastos
- Ter uma base sólida para evoluir depois (relatórios, autenticação, etc.)
